## Run Locally
Clone the project
```bash
  https://github.com/Asanda001019/TASK4-UI.git
```
Go to the project directory
```bash
  cd TASK6-EMPLOYEES-APPLICATION/TASK4-UI
```
Install dependencies
```bash
npm install 
```
Start the react server
```bash
npm run dev
````

## Tech Stack
**Client:** React JS, HTML5, JavaScript, CSS3
